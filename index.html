Mis proyectos

Proyecto 1

La calculadora tendrá una pantalla para mostrar números y resultados, y botones para los dígitos (0-9), las operaciones principales (suma, resta, multiplicación y división), un botón de igual para calcular el resultado y un botón para limpiar todo (C).

Proyecto 2

La aplicación tendrá una lista para mostrar las tareas pendientes, con campos para agregar nuevas tareas y botones que permitirán marcar cada tarea como completada. También habrá una opción para eliminar tareas que ya no sean necesarias, lo que facilitará la organización y gestión de las mismas.

Ver Proyecto

Proyecto 3

Un reloj digital que muestre la hora actual en tiempo real utilizando JavaScript. Este reloj se actualizará automáticamente cada segundo, asegurando que los usuarios siempre vean la hora precisa. Además, incluirá opciones para mostrar la hora en diferentes formatos, como 12 y 24 horas, y permitirá personalizar el estilo visual para adaptarse a las preferencias del usuario.

Ver Proyecto

Proyecto 4

Un botón interactivo que, al ser presionado, muestre frases motivadoras aleatorias. Cada vez que el usuario haga clic en el botón, se generará una nueva frase inspiradora, diseñada para elevar el ánimo y fomentar la positividad.
